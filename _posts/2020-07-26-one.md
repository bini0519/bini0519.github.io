Django란?
웹 앱 개발환경

설치 순서
1) 가상환경 만들기
python -m venv <이름>

2) 가상환경 실행하기
source myvenv/Scripts/activate

3) 장고 설치
pip install django

+) 가상환경 종료하기
diactivate
